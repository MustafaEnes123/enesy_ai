# Contributing to Enesy AI Assistant

First off, thank you for considering contributing to this project! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can You Contribute?

### Reporting Bugs
If you find a logic error, a hallucination loop, or a performance bottleneck, please open an issue. Include:
*   Your operating system and hardware specifications.
*   The exact prompt that caused the issue.
*   Steps to reproduce the behavior.

### Suggesting Enhancements
We are always looking to optimize our custom model. If you have ideas for better weight distribution or improved inference speeds, feel free to open a discussion or a pull request.

## Pull Request Process
1.  Fork the repository and create your branch from `main`.
2.  Make your changes and test them locally.
3.  Ensure your commit messages are clear and descriptive.
4.  Open a Pull Request with a detailed explanation of the changes and the problem they solve.

We will review your PR as soon as possible. Thank you for helping us build a better AI!
